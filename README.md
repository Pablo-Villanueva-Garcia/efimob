# EFIMOD PROJECT

## Requirements

- Docker
- npm

## Usage

    cd central-system-for-web
    npm install
    cd ..
    docker-compose up

## Endpoints

- frontend: http://localhost:3001/
- mongo-express: http://localhost:8080/
